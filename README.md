# NeoGuard AI — Neonatal Incubator Monitoring

AI-powered neonatal incubator monitoring system with real-time vitals tracking and risk prediction.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

```sh
# Step 1: Install dependencies
npm install

# Step 2: Start the development server
npm run dev
```

## Deployment

You can deploy this project using any modern web hosting service like Vercel, Netlify, or AWS Amplify.
